# 🌍 Language Translation Tool

## 📌 Description
This is a simple Python-based language translation tool that translates text into different languages using deep-translator.

## 🚀 Features
- Translate text into multiple languages
- Easy to use
- Command-line interface

## 🛠️ Technologies Used
- Python
- deep-translator

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python app.py

## 📷 Example
Input: Hello  
Output: नमस्ते
